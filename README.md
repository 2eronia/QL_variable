### 容器构建命令
```shell
docker run -dit \
  -p 自定义:5008 \
  -e TZ=Asia/Shanghai \
  --name qlva \
  --restart unless-stopped \
  xgzk/qlvariable:latest
```
```http request
http://IP:自定义端口/ 提交和更新页面
http://IP:自定义端口/log 日志页面
```
需要定时任务配置文件权限
